Wordy 隐私政策

最后更新日期：2025年12月11日

概述

Wordy（以下简称"本应用"）尊重并保护所有使用本应用的用户（以下简称"用户"或"您"）的隐私。本隐私政策说明了我们如何收集、使用、存储和保护您的个人信息。

信息收集

我们收集的信息

本应用收集以下类型的信息：

1. 学习数据

  • 词汇学习记录：您学习过的单词、掌握状态、复习时间
  • 造句练习记录：您提交的英文句子、AI 评估结果、练习时间戳
  • 学习进度统计：已见过单词数、已掌握单词数、学习中单词数

2. 应用设置

  • 用户偏好设置：语音设置（语速、发音类型）、iCloud 同步开关
  • 引导状态：是否已完成首次使用引导

3. 设备信息

  • 系统信息：iOS 版本（仅用于兼容性）
  • 应用版本：用于功能支持判断

我们不收集的信息

本应用不会收集以下信息：

  • 您的姓名、电话号码、电子邮件地址等个人身份信息
  • 您的地理位置信息
  • 您的通讯录、照片或其他设备数据
  • 您的设备标识符（IDFA）
  • 任何用于广告追踪的信息

信息使用

使用目的

我们收集的信息仅用于以下目的：

1. 提供核心功能

  • 展示单词卡片和词典内容
  • 记录学习进度和复习计划
  • 保存造句练习历史

2. AI 评估服务

  • 将您提交的英文句子发送到阿里云通义千问 API 进行评估
  • 接收并展示 AI 评分和反馈建议
  • 缓存评估结果以优化性能

3. 改善用户体验

  • 保存您的偏好设置（语音、同步等）
  • 提供个性化的学习体验

第三方服务

本应用使用以下第三方服务：

阿里云通义千问 API

  • 用途：AI 句子评估和反馈生成
  • 传输数据：您输入的英文句子、目标单词、场景信息
  • 数据处理：API 调用通过 HTTPS 加密传输

重要说明：

  • 我们不会向通义千问 API 发送任何个人身份信息
  • 仅发送必要的句子内容用于评估
  • API 调用需要您自行配置 API Key

数据存储

本地存储

所有学习数据默认存储在您的设备本地：

  • 使用 Apple SwiftData 框架进行数据持久化
  • 数据存储在应用沙盒中，其他应用无法访问
  • 受 iOS 系统级加密保护

iCloud 同步（可选）

如果您启用 iCloud 同步功能：

  • 数据将同步到您的 iCloud 私有数据库
  • 使用 Apple CloudKit 服务，端到端加密
  • 仅您本人可以访问同步的数据
  • 您可以随时在设置中禁用同步

同步的数据包括：

  • 词汇列表和学习进度
  • 造句练习记录
  • 应用设置和偏好

数据安全

我们采取以下措施保护您的数据安全：

  1. 加密传输：所有网络请求使用 HTTPS 加密
  2. 本地加密：数据受 iOS 系统级加密保护
  3. 访问控制：数据仅限应用内访问，其他应用无法读取
  4. 最小权限：应用仅请求必要的系统权限
  5. 无服务器存储：我们不在自己的服务器上存储您的任何数据

数据共享

我们不会与任何第三方共享、出售或出租您的个人信息，除非：

  • 获得您的明确同意
  • 法律法规要求
  • 保护我们的合法权益

您的权利

您对自己的数据拥有以下权利：

1. 访问权

您可以随时在应用内查看您的所有学习数据和练习记录。

2. 删除权

  • 删除单个记录：在应用内删除特定的练习记录
  • 删除所有数据：卸载应用将删除所有本地数据
  • 删除 iCloud 数据：在 iOS 设置中管理 iCloud 存储

3. 导出权

您可以通过以下方式导出数据：

  • 查看应用内的学习统计和历史记录
  • 截图保存重要信息

4. 控制权

  • 随时启用或禁用 iCloud 同步
  • 控制是否使用 AI 评估功能
  • 自定义应用设置和偏好

儿童隐私

本应用适合所有年龄段用户使用。我们不会故意收集 13 岁以下儿童的个人信息。如果您是家长或监护人，发现您的孩子向我们提供了个人信息，请联系我们，我们将及时删除相关信息。

Cookie 和追踪技术

本应用不使用 Cookie、网络信标或其他追踪技术。我们不会追踪您的应用使用行为或跨应用活动。

数据保留

  • 学习数据：永久保留，直到您删除或卸载应用
  • 练习记录：永久保留，直到您手动删除
  • 缓存数据：可能被系统自动清理
  • iCloud 数据：根据您的 iCloud 设置保留

隐私政策变更

我们可能会不时更新本隐私政策。更新后的政策将在应用内或通过其他适当方式通知您。继续使用本应用即表示您接受更新后的隐私政策。

重大变更通知：

  • 如果隐私政策发生重大变更，我们会在应用内显著位置通知您
  • 您可以选择接受新政策或停止使用应用

适用法律

本隐私政策受中华人民共和国法律管辖。如有争议，应首先通过友好协商解决。

其他重要信息

API Key 管理

  • 本应用需要您自行配置阿里云通义千问 API Key
  • API Key 存储在您的设备本地，使用 iOS Keychain 加密保护
  • 我们无法访问您的 API Key
  • 请妥善保管您的 API Key，不要与他人分享

开源信息

  • 本应用基于开源项目开发
  • 源代码可在 GitHub 上查看
  • 您可以审查代码以了解数据处理方式

数据安全建议

  • 定期备份重要的学习数据
  • 使用强密码保护您的 Apple ID
  • 启用双重认证保护 iCloud 账户
  • 不要在公共设备上使用本应用
